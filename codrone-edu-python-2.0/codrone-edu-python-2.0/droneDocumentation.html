Drone Class Documentation

Description: This class provides an interface for interacting with the CoDrone EDU drone. It includes functionalities for connecting to the drone, controlling its movement, accessing sensor data, managing LEDs and buzzers, and performing advanced flight sequences.

Initialization (__init__)

Description: Initializes the Drone object, setting up serial communication, receiver, event handlers, and other internal variables.

Parameters:

  • flagCheckBackground: Boolean, determines whether to automatically update data in the background. Default: True
  • flagShowErrorMessage: Boolean, determines whether to display error messages. Default: False
  • flagShowLogMessage: Boolean, determines whether to display log messages. Default: False
  • flagShowTransferData: Boolean, determines whether to display transferred data. Default: False
  • flagShowReceiveData: Boolean, determines whether to display received data. Default: False
  • swarm: Boolean, True if drone is part of a swarm, False otherwise. When True, asyncio.sleep is not overwritten. Default: False

Returns: None

Destructor (__del__)

Description: Releases the serial port if its still open when the object is destroyed. Disconnects the drone when object is destroyed.

Parameters: None

Returns: None
